# Become The Bank - Seller Financing Calculator

A fully self-contained 3-step wizard calculator for comparing traditional property sales with seller financing options.

## Features

- **Step 1**: Property details with simple/advanced tax calculations
- **Step 2**: Buyer capacity and affordability analysis
- **Step 3**: Side-by-side comparison of traditional vs seller financing

## For Webflow Integration

To embed in Webflow:

1. Open `calculator-embed.html`
2. Copy the `<style>` and `<script>` blocks
3. In Webflow: Page Settings → Custom Code → Before `</body>`
4. Paste both blocks
5. Add `<div id="become-the-bank-calculator"></div>` where you want the calculator

## Technology

- Vanilla JavaScript (no frameworks)
- Self-contained CSS
- No external dependencies
- Mobile responsive
